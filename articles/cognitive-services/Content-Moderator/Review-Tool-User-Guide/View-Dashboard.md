# <a name="view-dashboard"></a>Bekijk het dashboard

Klik op de **Dashboard** tabblad om te bekijken van belangrijke metrische gegevens voor alle inhoud controleren die binnen het hulpprogramma uitgevoerd. Totaal aantal voltooide en in uitvoering beoordelingen voor tekst en afbeeldingen, kunt u zien. U kunt metrische gegevens weergeven voor uw standaardteam en uw subteams.
