# <a name="workloads-on-azure-virtual-machines"></a>Workloads op Azure Virtual Machines

Virtuele machines in Azure ondersteunt veel werkbelastingen. Elke workload kunt informatie over het ontwerp van de optimale voor implementaties op virtuele machines in Azure zijn gespecialiseerd. Hier ziet u enkele van de richtlijnen die we voor specifieke werkbelastingen bieden kunnen.


- [SAP](./sap/get-started.md) in Azure kunt uitvoeren uw essentiële SAP workloads en scenario's op een schaalbaar, compatibel en zakelijk bewezen platform.
- [Oracle](./oracle/oracle-considerations.md) ondersteunt het uitvoeren van Oracle DB 12.1 Standard en Enterprise-edities op installatiekopieën voor virtuele machines in Azure op basis van Oracle Linux. 
- [Mainframe opnieuw hosten](./mainframe-rehosting/overview.md) 



**Volgende stappen**

Maak kennis met Azure-VM's met de virtuele machine van Azure-snelstartgidsen voor [Linux](../linux/quick-create-cli-nodejs.md) en [Windows](../windows/quick-create-powershell.md).
