# <a name="micro-focus-devtest-environments-on-azure"></a>Micro Focus dev/test-omgevingen op Azure

Aan de slag met Azure met een ontwikkelings- en -omgeving is een uitstekende manier om uw moderniseringspogingen te verbeteren. Mainframe-toepassingen die afhankelijk van verouderde systemen zoals IBM CICS (systeem voor klantinformatie klant) en IMS (Information Management System zijn) kunnen worden rehosted op Azure met behulp van oplossingen van partners.

Micro Focus is een partnerschap aangegaan met Microsoft om aantrekkelijke mainframe-hulpprogramma's voor migratie en het opnieuw hosten bieden. U kunt mainframe-toepassingen voor ontwikkeling, testen en productiescenario's uitvoeren op Azure en het Microsoft Windows-platform.

Micro Focus biedt ontwikkeling opnieuw hosten en test oplossingen die een uitbreiding vormen van bestaande COBOL en PL opnieuw hosten / ik business-toepassingen naar Azure.
