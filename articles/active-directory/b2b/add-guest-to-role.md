# <a name="grant-permissions-to-users-from-partner-organizations-in-your-azure-active-directory-tenant"></a>Machtigingen verlenen aan gebruikers van partnerorganisaties in uw Azure Active Directory-tenant

Gebruikers van Azure Active Directory (Azure AD) B2B-samenwerking als gastgebruikers ook kunnen worden toegevoegd aan de directory en machtigingen voor gasten in de map standaard worden beperkt. Uw bedrijf moet mogelijk bepaalde gastgebruikers om in te vullen hogere bevoegdheid rollen in uw organisatie.
