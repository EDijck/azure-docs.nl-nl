# <a name="container-publishing-prerequisites"></a>Vereisten voor het publiceren van container

Dit artikel beschrijft de vereisten voor het publiceren van een container-aanbieding op Azure Marketplace.  Als u hebt nog niet gedaan, wordt u aangeraden de [Publicatiehandleiding voor Containers bieden](../../marketplace-containers.md).

## <a name="publishing-prerequisites"></a>Vereisten voor publiceren

U hebt voor het publiceren van een nieuwe containerinstallatiekopie, om te voldoen aan de volgende vereisten:

- Toegang tot de Cloud Partner-Portal. Zie voor meer informatie, de Azure Marketplace en AppSource publicatiehandleiding voor.
- De voorwaarden voor Azure Marketplace-overeenkomst
- Host uw technische asset van de container in een Azure Container Registry.
- Klaar voor gebruik op de containermetagegevens van uw hebben. Bijvoorbeeld, de volgende niet-uitputtende lijst:
  - Een titel
  - Een beschrijving (in HTML-indeling)
  - Een logoafbeelding (in PNG-indeling) en in deze vaste grootte: 40, 40 px, 90 x 90 px, 115 x 115 px en 255 x 115 px.
- Een *gebruiksvoorwaarden* en een *privacybeleid* instructies
- Documentatie voor de container-oplossing
- Contactpersonen voor ondersteuning

## <a name="next-steps"></a>Volgende stappen

Als u hebt nog niet gedaan, moet u [voorbereiden van uw container technische activa](./cpp-create-technical-assets.md) voordat u [maken van uw aanbieding container](./cpp-create-offer.md).
