# <a name="saas-application-publishing-prerequisites"></a>Vereisten voor SaaS-toepassing publiceren

Dit artikel beschrijft de vereisten voor het publiceren van een SaaS-toepassing (App)-aanbieding op Azure Marketplace.  Als u hebt nog niet gedaan, raadpleegt u de [SaaS-toepassingen bieden Publicatiehandleiding voor](../../marketplace-saas-applications-technical-publishing-guide.md). 


## <a name="publishing-prerequisites"></a>Vereisten voor publiceren

Als u een nieuwe SaaS-App-aanbieding publiceren, moet u voldoen aan de volgende vereisten:
- Toegang tot de [Cloud Partner-Portal](https://cloudpartner.azure.com/).  De Cloud Partner-Portal biedt toegang op basis van rollen, zodat meerdere personen samen te werken voor het publiceren van een aanbieding. 
- Voordat u een aanbieding kan worden gepubliceerd namens een publisher-account, moet een van de personen met de rol van eigenaar gaat ermee akkoord om te voldoen aan de [gebruiksvoorwaarden](https://azure.microsoft.com/support/legal/website-terms-of-use/), [privacyverklaring van Microsoft](https://www.microsoft.com/privacystatement/default.aspx), en [ Microsoft Azure Certified-Programmaovereenkomst](https://azure.microsoft.com/support/legal/marketplace/certified-program-agreement/).
- Alle niet-Engelse inhoud moet vergezeld gaan van een Engelse versie. Deze inhoud bevat Storefront tekst, documenten, schermafbeeldingen, gebruiksvoorwaarden en het privacybeleid. Het is aanvaardbaar is voor een URL handige koppeling naar een niet-Engelse-toepassing. 
- Uw metagegevens klaar voor gebruik hebben. De volgende (niet volledige) lijst toont een voorbeeld van deze metagegevens:
  - Een titel
  - Een beschrijving (in HTML-indeling)
  - Een logoafbeelding (in PNG-indeling) en in deze vaste grootte: 40, 40 pixels, 90 x 90 pixels, 115 x 115 pixels en 255 x 115 pixels.
  - Een *gebruiksvoorwaarden* en *privacybeleid* instructies
  - Documentatie voor Application
  - Contactpersonen voor ondersteuning


## <a name="next-steps"></a>Volgende stappen

Nadat u deze vereisten wordt voldaan, kunt u [maken van uw SaaS-aanbieding](./cpp-create-offer.md).
