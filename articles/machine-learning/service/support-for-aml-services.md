## <a name="submit-doc-feedback"></a>Document feedback verzenden 

U kunt **aanvragen indienen** voor extra materiaal met behulp van de **inhoud feedback** knop aan het einde van elk artikel.

## <a name="get-service-support"></a>Ondersteuning voor service 

Bekijk deze bronnen voor ondersteuning:

+ **Technische ondersteuning voor Azure-klanten**: [Indienen en beheren van ondersteuningsaanvragen](https://docs.microsoft.com/azure/azure-supportability/how-to-create-azure-support-request) via Azure portal.

+ **-Gebruikersforum**: Vragen stellen, vragen te beantwoorden en verbinding maken met andere gebruikers in de [ondersteuningsforum voor Azure Machine Learning-service op MSDN](https://aka.ms/aml-forum-service).

+ **Stack Overflow**: Ga naar de Azure Machine Learning-community op [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) gelabeld met 'Azure-Machine-Learning'.

+ **Delen van suggesties** en functies aanvragen in onze [Azure Machine Learning Feedback Channel](https://feedback.azure.com/forums/257792-machine-learning). Selecteer de **productfeedback** knop aan het einde van elk artikel om te delen kunt kiezen.
