## <a name="set-expiration"></a>Vervaldatum instellen

Het is ook mogelijk om te configureren van het anker om op een bepaald tijdstip in de toekomst automatisch laten verlopen. Wanneer een anker is verlopen, wordt deze niet meer zich bevindt of bijgewerkt. Vervaldatum kan alleen worden ingesteld wanneer het anker wordt gemaakt. Vervaldatum later bijwerken is niet mogelijk. U kunt dus de verlopen voordat u opslaat in de cloud instellen.
